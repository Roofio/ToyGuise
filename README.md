# ToyGuise

A World of Warcraft addon that displays a clean, clickable grid of your owned **transformation and disguise toys**.

Click any toy to use it instantly — perfect for quick costume changes, roleplay, or fun!

### Features
- Shows only **usable** transformation/disguise toys
- Minimap icon

### Installation
1. Download the latest release
2. Unzip the `ToyGuise` folder into your  
   `World of Warcraft\_retail_\Interface\AddOns\` directory
3. Restart WoW or reload UI with `/reload`

### Usage
- Click the minimap button or type `/tg` to open/close the grid
- Click any toy icon to use it

### Commands
- `/tg` or `/toyguise` — Toggle the window
- `/tg perf` — Show addon memory and CPU usage

### Notes
- Toy list is curated in a separate file (`ToyGuise_ToyList.lua`). If recommending more toys to add, pull requests should include the item id and name and in item id order.
